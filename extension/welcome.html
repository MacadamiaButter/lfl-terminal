lfl-terminal is installed

A command terminal for any page, plus a local model that proposes page actions you approve.

Open it

  • Press the backtick key ` (when focus isn't in a page input)
  • Press Ctrl+K
  • Click the toolbar button

Esc closes it again.

Get started

Once it's open, type help for the full command list, or tour for a 60-second walkthrough of the deterministic commands, the model lane, and the approval card.

What it does

Deterministic commands (ls, open, find, read, and more) run locally, no model involved. Type something else and a local model proposes ONE page action - click, fill, navigate - which you approve or reject before anything happens.

Privacy: deterministic commands run entirely in your browser. Model requests go only to your bridge at 127.0.0.1:1238 - self-hosted, nothing leaves your machine; on the shared beta the bridge relays them to the cohort model over the private tailnet, never the open internet. Page actions always need your explicit approval; the model never acts on its own.
